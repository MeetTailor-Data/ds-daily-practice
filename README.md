# 30 Days of Data Science Mini Projects

A collection of 30 beginner-friendly Python data science projects,
each under 20 lines of code with at least one visualization.

---

## Goal

- Build a daily GitHub contribution streak
- Practice core Python and Data Science libraries
- Create a visible, growing portfolio

---

## Libraries Used

- Python 3.x
- Matplotlib
- Seaborn
- NumPy
- Pandas

---

## Project List

| Day | Project Name | Chart Type |
|-----|--------------|------------|
| 01  | Age Distribution of Students | Histogram |
| 02  | Monthly Expenses Tracker | Line Chart |
| 03  | Favorite Fruits Survey | Pie Chart |
| 04  | Student Marks Bar Chart | Bar Chart |
| 05  | Temperature of a Week | Line Chart |
| 06  | Runs Scored by Batsmen | Horizontal Bar |

## How to Run

1. Clone this repository
git clone https://github.com/MeetTailor-Data/30-days-ds-projects.git

2. Install required libraries
pip install matplotlib seaborn numpy pandas

3. Run any project
python day01_age_distribution.py
